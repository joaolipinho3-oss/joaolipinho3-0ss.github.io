# Simple Counter

Um contador simples desenvolvido com HTML, CSS e JavaScript.

## 📋 Funcionalidades

- Incrementar o contador
- Decrementar o contador
- Reiniciar o contador
- Atalhos de teclado para controle rápido
- Interface simples e intuitiva

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Estrutura do Projeto

```
simple-counter/
│
├── index.html
├── styles.css
├── script.js
└── README.md
```

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/simple-counter.git
```

2. Acesse a pasta do projeto:

```bash
cd simple-counter
```

3. Abra o arquivo `index.html` em qualquer navegador.

## ⌨️ Atalhos de Teclado

| Tecla | Ação |
|--------|--------|
| ArrowUp | Incrementa o contador |
| ArrowDown | Decrementa o contador |
| R | Reinicia o contador |
| Spacebar | Adiciona o valor do incremento (Apenas desktop) |

> Os atalhos podem variar de acordo com a implementação atual do projeto.
